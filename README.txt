NASApy


This application presents the latest news or articles from NASA in a simple and fast GUI. It features a clickable link to view the latest image from space. Utilizing NASA's official API, this app is your window to the wonders of space. Enjoy exploring the cosmos!


Setup and Run
1. **Install Python**: Ensure Python is installed on your system. You can download it from (https://www.python.org/downloads/). Python comes with the `tkinter` and `webbrowser` modules, so no additional installation is required for them.


2. Install Dependencies: Install the `requests` library, which is necessary to fetch data from NASA's API. Use the command:


   pip install requests


3. Run the Script: Execute the script to view the latest news and image of space from NASA.
